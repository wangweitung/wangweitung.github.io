# 30岁对自己的思考

&emsp;&emsp;已经30岁了，宝宝也有了，目前公司已经开始裁员了，我的核心竞争力在哪里？我的能力在哪里？如何才能让未来的自己位于不败之地？虽然每次都在问自己这个问题，但是真的没有沉下心来去行动。

&emsp;&emsp;思考过，但是没有行动的时候：

```
1.在2017年8月31日离开MC时。
2.在2018年12月24日离开YT，并且和YT的同事谈心时。
3.在2020年不知道什么时候被LN裁员时。
```

&emsp;&emsp;所以，这次我该怎么做？继续这样吗？还是有所改变？

&emsp;&emsp;昨天晚上凌晨3点，我突然迷迷糊糊的意识到，“*祸兮福所倚*，*福兮祸所伏*”，或许在我30岁时，遇到这样的变革并不是坏事，更让我知道了应该做什么，思考什么，找到自己的核心竞争力。

&emsp;&emsp;目前的选择无非就是两个：

```
1、留在LN，继续面临这样的低产期，在随后的3-5年熬过车市的寒冬，然后继续在WH工作和生活。
2、离开LN，找到自己喜欢的工作。
```

&emsp;&emsp;在此过程中，进行升级转型，抓住自己对代码的喜好，并且结合自己的7年的工作经验，转为WLFZ。在未来智能制造的过程中，不再让自己被淘汰。
&emsp;&emsp;可是第二个选项面临了太多的艰难和挑战。前途又是这么的扑朔迷离。
&emsp;&emsp;真的该好好做一做SMART分析了。好让自己坚定决心，一个心思的往前走。

&emsp;&emsp;因此目前的做法如下：

 ```
1. 继续等待公司政策。
2. 更新简历，在猎聘上查找新的工作。
3. 如果LN的竞聘没有上，先接受公司的待岗，那么有约1-3个月的时间寻找新的工作。
4. 如果LN的竞聘上了，那么不论如何也不能安于现状，得过且过了。得要继续努力，为自己的40岁做打算，没有人知道，40岁时，会不会再遇到类似的情况。
5. 在此期间，努力学习C语言，plant simulation，为自己的新工作打基础。
 ```


   &emsp;&emsp;希望自己在2020年能有新的起点，新的收获。
   &emsp;&emsp;最后，放上一张LN桌面，聊做纪念，顺便警醒着自己不断冲刺向前。
