十、文章状态在 Hexo 资料夹中的架构

  所有的文章包含未发布、已发布、已删除的文章，都会以不同资料夹做区分，被放在 /Hexo资料夹/source 之下(如下图)。要注意的是，虽然从后台删除了文章，但其实文章还是会被存放在 /Hexo资料夹/source/_discarded 中。

  另外，文章的档案是以“连结网址名称”来命名，所以如果要修改文章连结，后台是无法修改的，需直接修改档名。

  参考链接：https://ed521.github.io/2019/08/hexo-admin/
